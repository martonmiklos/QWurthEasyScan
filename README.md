

QWurthEasyScan
================================

A small graphical utility to read and save barcodes captured with the [WÃ¼rth EasyScan](http://www.servicesupply.com/web/en/wis/geschaeftsbereiche/cps_ecommerce/scannergesttztebestellsysteme/scannen.php) barcode readers.

Written in C++ using the Qt5 framework.
For the serial communication the [qextserialport library](http://code.google.com/p/qextserialport/) library was used.
